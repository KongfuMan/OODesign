## requirements
给一个四向十字路口，有sensor会在车辆进入sensor和出sensor的时候发出event，sensor可以发信息给路口摄像机拍照，有红绿灯，有可用的车辆图片查询的API（返回车辆信息）。
设计一个系统给警察局让警察局有效监控车辆行车规范。

```
+-----------+      +------------+      +---------+
|   Green   +----->+   Yellow   +----->+   Red   |
+-----+-----+      +------------+      +----+----+
      ^                                     |
      |                                     |
      +-------------------------------------+
      
```
